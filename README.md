# D Source Network Portfolio

**Live website:** https://Dsource3101.github.io/dsourcenetworks/

## Overview
This repository hosts the complete single‑page portfolio for D Source Network, including:
- Premium corporate gifting showcase
- Customer login/signup (demo mode)
- **Employee portal** with secure login, request‑access form, attendance, chat, and inventory management – all embedded in the same page.

## How to Run Locally
```bash
# Clone the repo
git clone https://github.com/Dsource3101/dsourcenetworks.git
cd dsourcenetworks
# Open index.html in a browser
# No build step – pure HTML/CSS/JS
```

## Deployment
The site is automatically published via GitHub Pages. After any change, run the provided `Upload_To_GitHub.bat` script (or the equivalent git commands) to push updates.
